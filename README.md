# Scenerygram

A social app (like Instagram) for sharing picturesque scenery. Built for fun.

Photos courtesy of [Unsplash](https://unsplash.com/).

## Demo

You can go to the [site](https://scenerygram.vercel.app/) and use the following test credentials:

```

```

## Tech stack

Client:
- React
- TypeScript
- Context API
- Tailwind CSS
- GraphQL
- Apollo Client
- React Query
- React Hook Form
- Cloudinary
- Google Places API
- Vite
- Vercel

Server:

- Node.js
- Express
- TypeScript
- GraphQL
- Apollo Server
- TypeORM
- PostgreSQL
- Fly.io
- Neon
